### Screens and Goals

- **Calendar Month**: show colored days; click a day → list transactions.
- **Calendar Week**: same as month view, but with 7-day focus.
- **Day View**: list events; add transactions; view projected end-of-day balance.
- **Transactions**: filter, create, edit, delete.
- **Categories**: create, edit, assign parent.
- **Settings → Accessibility**: font scale, high contrast, narration toggle.
